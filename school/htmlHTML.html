Minecraft

Waarom minecraft?

Ik heb minecraft gekozen omdat ik de game een lange tijd heb gespeeld in mijn jongere jaren.
heb hier veel leuke herringeringen mee beleefd en daarom schrijf ik erover. De nederlandse versie van Minecraft kwam uit in 2011.

In minecraft moet je overleven, je hebt verschillende gebieden die je kan ontdekken, uiteindelijk is het je doel om de Enderdragon te verslaan.
Je kan deze game spelen met vrienden, maar ook kan je het alleen spelen. Er zijn heel veel servers waar je op kunt spelen.
Je hebt verschillende speltypen die je online kan spelen op servers. Deze worden allemaal gemaakt door de community zelf.
